##start
0. 进入MyNpm 执行npm install
1. 安装应用
2. 执行 adb reverse tcp:8081 tcp:8081
3. 进入MyNpm： 执行npm start
4. 进入应用授予获取运动权限等，摇一摇即可进入开发者选项，设置Enable live load.
